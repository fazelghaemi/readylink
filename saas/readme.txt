با تشکر از انتخاب اسکریپت RDYS. این سیستم یک پلتفرم کامل مدیریت لینک (SaaS) است که امکاناتی نظیر ثبت‌نام کاربر، اتصال دامنه اختصاصی، آمار پیشرفته و API را فراهم می‌کند.

-------------------------------------------------------------------
1. حداقل پیش‌نیازهای سرور
-------------------------------------------------------------------
- PHP نسخه 7.4 یا بالاتر
- دیتابیس MySQL یا MariaDB
- وب‌سرور Apache یا LiteSpeed (پشتیبانی از mod_rewrite برای فایل .htaccess)
- فعال بودن اکستنشن PDO_MySQL در PHP

-------------------------------------------------------------------
2. راهنمای نصب سریع (کمتر از ۲ دقیقه)
-------------------------------------------------------------------
۱. ساخت دیتابیس:
   وارد کنترل پنل هاست خود (cPanel/DirectAdmin) شوید و یک دیتابیس جدید و یک یوزر دیتابیس بسازید.

۲. ویرایش کانفیگ:
   فایل `config.php` را با یک ویرایشگر متن باز کنید و مقادیر زیر را طبق دیتابیس خود تغییر دهید:
   - DB_HOST (معمولاً localhost)
   - DB_NAME (نام دیتابیس)
   - DB_USER (نام کاربری دیتابیس)
   - DB_PASS (رمز عبور دیتابیس)
   * نکته: آدرس سایت (SITE_URL) به صورت خودکار تشخیص داده می‌شود، اما می‌توانید دستی هم وارد کنید.

۳. آپلود فایل‌ها:
   تمام فایل‌های موجود در پوشه پروژه را در پوشه اصلی هاست خود (`public_html`) آپلود کنید.
   * نکته مهم: فایل‌های `.htaccess` و تصاویر (`assets`) حتما باید آپلود شوند.

۴. اجرای نصب‌کننده:
   مرورگر را باز کنید و به آدرس سایت خود به صورت زیر بروید:
   [https://your-domain.com/install.php](https://your-domain.com/install.php)

   روی دکمه "نصب و راه‌اندازی" کلیک کنید. سیستم به صورت خودکار جداول را می‌سازد.

۵. پایان:
   پس از مشاهده پیام موفقیت، حتماً فایل `install.php` را از روی هاست پاک کنید.

-------------------------------------------------------------------
3. اطلاعات ورود پیش‌فرض (مدیر کل)
-------------------------------------------------------------------
نام کاربری: admin
رمز عبور: admin

* هشدار: بلافاصله پس از اولین ورود، از طریق دیتابیس یا فایل reset_password.php رمز خود را تغییر دهید.

-------------------------------------------------------------------
4. لیست فایل‌های سیستم
-------------------------------------------------------------------
- index.php        (صفحه اصلی و لندینگ پیج)
- admin.php        (پنل مدیریت پیشرفته و داشبورد کاربران)
- login.php        (صفحه ورود امن)
- register.php     (صفحه ثبت‌نام کاربران)
- api.php          (موتور پردازش درخواست‌ها و API)
- redirect.php     (موتور هدایت‌کننده لینک‌ها)
- functions.php    (توابع هسته و منطق سیستم)
- config.php       (تنظیمات دیتابیس)
- install.php      (نصب‌کننده و تعمیرکار دیتابیس - باید حذف شود)
- export.php       (خروجی اکسل)
- 404.php          (صفحه خطای اختصاصی)
- style.css        (استایل‌ها و تم رِدی استودیو)
- sw.js            (سرویس ورکر برای PWA)
- manifest.json    (مانیفست اپلیکیشن وب)
- .htaccess        (تنظیمات سرور و روتینگ)
- *.png/webp/woff  (فایل‌های گرافیکی و فونت)

-------------------------------------------------------------------
5. اطلاعات پلن‌های کاربری (Plans)
-------------------------------------------------------------------
سیستم دارای ۳ پلن پیش‌فرض در دیتابیس (جدول plans) است:
ID 1: رایگان (پیش‌فرض برای ثبت‌نام کنندگان) - ۵۰ لینک
ID 2: حرفه‌ای - ۵۰۰۰ لینک + ۳ دامنه + دسترسی API
ID 3: نامحدود (مخصوص ادمین) - دسترسی کامل

* برای تغییر محدودیت‌ها یا قیمت‌ها، جدول `plans` را در دیتابیس ویرایش کنید.
* برای ارتقای کاربر، مقدار `plan_id` را در جدول `users` تغییر دهید.

-------------------------------------------------------------------
6. اتصال دامنه اختصاصی (Addon Domain)
-------------------------------------------------------------------
کاربران می‌توانند دامنه‌های خود را به سیستم متصل کنند. برای این کار:
۱. دامنه جدید باید به هاست شما (از طریق A Record یا DNS) متصل شده باشد.
۲. دامنه باید روی پوشه نصب اسکریپت پارک شده باشد (Parked Domain / Alias).
۳. کاربر دامنه را در پنل "مدیریت دامنه‌ها" اضافه می‌کند.
